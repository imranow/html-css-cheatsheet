HTML Reference Table

Tag Name Description
< table > Table Defines a table.
< thead > Table Head Groups the header content in a table.
< tbody > Table Body Used to group the body content in sn HTML table.
< tr > Table Row Defines a row for a table.
< td > Table Data Cell Defines a cell of a table that contains data.

CSS Reference Table

Attribute Description
border Allows to specify the width, style, and color of an element's borders.
boarder-collapse Sets whether the table borders should be collapsed into a single border.
box-shadow Attaches one or more shadows to an element. Required values are h-offset and v-offset.
text-decoration Just as it sound, it specifies the decoration to text.
transform Lets you rotate, scale, skew, or translate an element.
transition A shorthand property for setting the four transition properties into a single property
display: flex Changes an element to a block-level container with flex items inside of it.
display: inline-flex Allows multiple flex containers to appear inline with each other.
justify-content Used to space items along the main axis.
align-items Used to space items along the cross axis.
flex-grow Used to specify how much space (and in what proportions) flex items absorb along the main axis.
flex-shrink Used to specify how much flex items shrink and in what proportions along the main axis.
flex-basis Used to specify the initial size of an element styled with flex-grow and/or flex-shrink.
flex Used to specify flex-grow, flex-shrink, and flex-basis in one declaration.
flex-wrap Specifies that elements should shift along the cross axis if the flex container is not large enough.
align-content Used to space rows along the cross axis.
flex-direction Used to specify the main and cross axes.
flex-flow used to specify flex-wrap and flex-direction in one declaration.
